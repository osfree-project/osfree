To add a new widget to the &xcenter;, right-click on an empty space on the &xcenter; bar and select "Add widget". A submenu appears which lists all available widget types. Select one from that list, and a new widget will appear in the &xcenter;.

The contents of that menu will depend on whether widgets have already been added to the &xcenter; and how many plug-in DLLs are contained in the plugins\xcenter directory of the &xwp; installation directory.

The exception to this rule are the "object button" widgets, which represent any Desktop object. These widgets can be used to display the contents of a folder (similar to XFolder's folder content menus), if the object is a folder, or to quick-start an object in its default view, if the object is not a folder.

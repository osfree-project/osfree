Classes: XWPSetup, XWPScreen
&xwp; also implements "screen border objects", which means that you can configure the &xwp; PM hook to automatically open a certain object in its default view if the mouse pointer is moved to one of the four corners or one of the four borders of your screen. You can also configure the &xwp; PM hook to initiate desktop switching in such cases if &pgr; is enabled.

For example, on my system, I have configured this feature to automatically open the "Drives" folder when I move the mouse pointer to the top-right corner of the screen.

In addition to opening objects, &xwp; can also display the PM window list with this feature.

Note: That page is only visible if the XWPScreen class is installed and the &xwp; hook has been enabled in "&xwp; Setup".

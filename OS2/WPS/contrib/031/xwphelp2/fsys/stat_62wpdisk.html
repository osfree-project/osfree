Schijf objecten (WPDisk class, b.v. de objecten in de &link_drivesfdr;) ondersteunen de volgende codes als uitbreiding op de codes voor alle objecten:
$fX
vrije ruimte op het station
$zX
totale ruimte op het station

In de bovenstaande twee situaties, is X een speciaal opmaak specificatie teken.

$F
bestandssysteem type (FAT, HPFS, JFS, CDFS, ...)
$L
stationslabel

Opmerkingen:

De $f en $z codes zijn opnieuw gedefinieerd voor schijf objecten. Normaal tonen zij de ruimte op het station waarop de map zich bevint, welke voor de "Stations" map, altijd de zelfde waarde is voor het station waarop de Werkplek zich bevindt. Zodoende is deze code geherdefineerd voor schijf objecten.

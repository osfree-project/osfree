Dit bericht wordt getoond wanneer u heeft geprobeerd een object te verwijderen, maar de &xwp; Prullenbak werd niet gevonden.

Dit kan de volgende redenen hebben:

  • De prullenbak is verwijderd. Dit is niet eenvoudig aangezien het prullenbak object zichzelf tegen verwijderen beschermd, maar toch het is mogelijk.

  • De prullenbak is zijn object ID kwijt geraakt. Om de prullenbak te vinden, zoekt &xwp; naar een object met het ID <XWP_TRASHCAN>.

  • Een ander object welke niet de &xwp; prullenbak is heeft het object ID <XWP_TRASHCAN>.
In ieder geval is het veilig om op "Ja" te drukken om een nieuwe prullenbak op uw werkplek te maken

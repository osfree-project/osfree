Op deze notitieblok pagina kunt u enige overzichtsinstellingen individueel configureren voor iedere map. Dit staat u toe om enige algemene instellingen ongedaan te maken die zijn gespecificeerd in het "Workplace Shell" object. Aanvankelijk zijn de instellingen hetzelfde als bij de algemene instellingen; als u hier iets wijzigt, zal dat alleen voor deze map worden vastgehouden, en hebben veranderingen in de algemene instellingen geen effect meer op deze map.

Wijzigingen gaan direct in.

  • Als een instelling is aangevinkt zal de respectievelijke instelling altijd voor deze map zijn ingeschakeld, onafhankelijk van wat u opgeeft in de algemene instellingen
  • Omgekeerd, als een instelling niet is aangevinkt, is de respectievelijke instelling altijd uitgeschakeld voor die map.
Druk op "Opheffen" om de instellingen die in gebruik waren voor u deze map opende weer terug te zetten.

Druk op "Standaard" om deze map zich te laten gedragen overeenkomstig de algemene instellingen in het "Workplace Shell" object. Dat zal de gegevens die voor deze map werden onthouden verwijderen.

Selecteer voor een gedetailleerde verklaring van elk veld uit de onderstaande lijst:

  • Favoriete map
  • Snel openen bij Werkplek start
  • Toon volledig pad in map titel
  • Voeg "Uitlijnen op raster" toe
  • Reageer op map sneltoetsen
  • Toon statusbalk
Extra informatie kan op de corresponderende hulp schermen voor de respectievelijke pagina's worden gevonden in het "Workplace Shell" object.

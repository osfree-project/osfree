&os2; heeft ingebouwde ondersteuning voor programma's die zijn geschreven voor Microsoft Windows 3.x. Deze ondersteuning wordt &winos2; genoemd en is een aangepaste versie van Microsoft Windows 3.1. dat op uw &os2; systeem draait. Dit stelt u in staat om Windows programma's uit te voeren alsof het &os2; programma's waren. B.v., u kunt Windows programma's draaien op de &os2; Werkplek of in een aparte volledig scherm sessie en steeds informatie delen tussen &os2; en Windows programma's.

Merk op dat de &winos2; ondersteuning equivalent is aan Windows 3.1, maar niet aan Windows 95. &winos2; ondersteunt geen 32 bits Windows applicaties. Om 32 bits Windows applicaties te draaien dient u uitbreidingen van derden te installeren.

In de &link_syssetupfdr;, vindt u het "&winos2setup;" object dat u in staat stelt algemene gegevens voor het gehele &winos2; subsysteem in te stellen. Deze instellingen worden gebruikt tenzij u een programma object heeft gemaakt dat wijst naar een &winos2; applicatie die speciale instellingen gebruikt.

Een &winos2; programma kan in een venster of in een volledig scherm draaien.

  • Alle &winos2; venster programma's delen standaard dezelfde sessie. M.a.w., zelfs als zij in meerdere vensters op de Werkplek verschijnen, draaien zij intern in hetzelfde proces. Alle programma's die in deze sessie draaien delen dezelfde DOS en &winos2; kenmerken die door het eerste programma in de sessie werden ingesteld.

    Echter, u kunt er voor kiezen dat een &winos2; programma een aparte sessie moet gebruiken door de respectievelijke instellingen uit te schakelen in het kenmerken notitieblok van het programma object. Dit heeft meer geheugen nodig maar zal het programma beschermen tegen andere &winos2; programma's omdat het in een apart proces draait.

  • Als u een &winos2; volledig scherm sessie draait, is de Werkplek niet langer zichtbaar. Echter, in een &winos2; volledig scherm sessie kunt u terugschakelen naar de &os2; Werkplek door het pictogram "Werkplek" te selecteren onderin de &winos2; sessie.
Als toevoeging, iedere &winos2; sessie (venster of volledig scherm) kan in de "standaard" of "uitgebreide compatibiliteits" modus draaien.
  • Standaard modus draait programma's die geschreven zijn voor Windows 3.0 en 3.1 standaard modus. 3.0 and 3.1 standard mode. -->

  • Uitgebreide compatibiliteits modus draait programma's die Microsoft Windows 3.0 en 3.1 uitgebreide modus nodig hebben.

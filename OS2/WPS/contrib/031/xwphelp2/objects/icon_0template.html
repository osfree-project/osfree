De "Model" keuze bepaalt of dit object een model is.

Het object wordt een model als u de "Model" keuze selecteert. Het pictogram van het object veranderd dan, en u kunt dan een kopie slepen van het object om een ander object te cre‰ren dat precies dezelfde instellingen krijgt als het model object. U dient deze instelling alleen te kiezen als u dit object exclusief wilt gebruiken om andere objecten van te maken.

&os2; heeft een aantal standaard model objecten in de &link_tempsfdr; die u de mogelijkheid geven om nieuwe objecten te maken van de meest voorkomende objecttypes. Het is echter niet noodzakelijk dat alle modellen in deze "Model" map staan, modellen kunnen overal op het systeem staan.

Als de "Model" instelling gedeselecteerd is, gedraagt het object zich als normaal.

Wees er op bedacht dat de "Model" instelling gedeselecteerd zal zijn voor bepaalde objecttypes die niet gedupliceerd kunnen worden.

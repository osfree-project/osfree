Het "Huidig pictogram" veld laat het pictogram zien (plaatje) dat momenteel gebruikt wordt om het geopende object te laten zien. &link_folders;.

Tenzij u het objectpictogram gewijzigd heeft, wordt het standaard pictogram normaliter afgeleid van het object zijn class. Anders gezegd, de meeste objecten hebben een standaard pictogram. Maar, u kunt elk pictogram vervangen op deze pagina.

  • Druk op een van de knoppen naast het pictogram scherm.
De volgende knoppen zijn beschikbaar:
  • "Bewerk pictogram" zal de &os2; Icon Editor opstarten met het huidige pictogram van het object. U kunt dan het pictogram aanpassen. Als u het pictogram opslaat vanuit de Icon Editor, zal het object de wijziging meenemen.

  • "Bladeren" zal een standaard &os2; bestandsdialoog openen die u de mogelijkheid geeft om een nieuw .ICO bestand te zoeken om voor dit object te gebruiken.

  • "Herstel pictogram" zal het pictogram van het object terugzetten naar zijn standaard, dat bepaald wordt door zijn class. De knop is uitgeschakeld als het object al het standaard pictogram heeft.

WPProgram is de WPS class voor alle &link_progobjs; (zie WPS classes voor details). In aanvulling op de instellingen teksten die herkend worden door de WPObject class, herkent WPProgram ook de volgende teksten:

ASSOCFILTER=filters

Stelt filters voor bestanden in om gekoppeld te worden met een programma object. Meerdere filters worden gescheiden door komma's.

Let op: Als &xwp; zijn uitgebreide koppelingen ingeschakeld heeft, wordt het filter opgeslagen in de INI bestanden, maar de aanpassing heeft dan geen effect voordat de uitgebreide koppelingen weer uitgeschakeld zijn omdat &xwp; volledig lak heeft aan WPS koppelingen via filters.

ASSOCTYPE=types

Stelt types voor bestanden in om gekoppeld te worden met een programma object. Meerdere types worden gescheiden door komma's.

EXENAME=bestandsnaam

Stelt het uitvoerbare programma in waarnaar het programma object moet verwijzen. Dit moet ofwel volledig gekwalificeerd zijn (b.v. neem het hele pad van het uitvoerbare programma) of een uitvoerbaar programma zijn op het PATH, anders zal het uitvoerbare programma niet werken.

MAXIMIZED={YES|NO}

Bepaald of het uitvoerbare programma waarnaar verwezen wordt door dit programma object gemaximaliseerd gestart moet worden. Dit heeft normaliter geen invloed op PM programma's.

MINIMIZED={YES|NO}

Bepaald of het uitvoerbare programma waarnaar verwezen wordt door dit programma object geminimaliseerd gestart moet worden. Dit heeft normaliter geen invloed op PM programma's.

PROGTYPE=type

Stelt expliciet het sessie type in voor het uitvoerbare programma waarnaar verwezen wordt door het programma object. type moet een van de volgende zijn:
  • DOSMODE: losstaande DOS mode.
  • FULLSCREEN: &os2; volledig scherm mode.
  • PM: Presentation Manager (PM).
  • PROG_30_STD: &winos2; standaard compatibiliteit volledig scherm mode.
  • PROG_31_ENH: &winos2; uitgebreide compatibiliteit volledig scherm mode.
  • PROG_31_ENHSEAMLESSCOMMON: &winos2; venster in de &winos2; uitgebreide compatibiliteit naadloze sessie.
  • PROG_31_ENHSEAMLESSVDM: &winos2; venster in een aparte sessie uitgebreide compatibiliteit mode.
  • PROG_31_STD: standaard compatibiliteit volledig scherm mode.
  • PROG_31_STDSEAMLESSCOMMON: &winos2; venster in de &winos2; standaard compatibiliteit naadloze sessie.
  • PROG_31_STDSEAMLESSVDM: &winos2; venster in een aparte sessie standaard compatibiliteit mode.
  • SEPARATEWIN: &winos2; venster dat draait in een aparte VDM.
  • VDM: DOS volledig scherm.
  • WIN: &winos2; volledig scherm.
  • WINDOWABLEVIO: &os2; venster.
  • WINDOWEDVDM: DOS venster.
  • WINDOWEDWIN: &winos2; venster.

NOAUTOCLOSE={YES|NO}

Bepaald of de sessie open moet blijven als het programma afgelopen is. Let er op dat dit geen invloed heeft op PM programma's. De standaard waarde is NO.

PARAMETERS=parameters

Stelt de parameters in die meegegeven moeten worden aan het uitvoerbare programma waarnaar het programma object verwijst. Dit mag de standaard programma object vervangingsmacro's bevatten.

SET variabele=waarde

Stelt een omgevingsvariabele in voor de sessie van het uitvoerbare programma. Wees er op bedacht dat &os2; ook omgevingsvariabelen gebruikt om DOS instellingen door te geven aan DOS en &winos2; sessies.

STARTUPDIR=pad

Stelt de werkdirectory bij het opstarten in voor de sessie.

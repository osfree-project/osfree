XWPTrashCan is de WPS class voor het &xwp; prullenbak object (zie WPS classes voor details). Omdat de prullenbak gewoon een bijzondere map is, worden naast de onderstaande XWPTrashCan teksten, ook de teksten van WPObject en WPFolder classes ondersteund.

Om het prullenbak object te krijgen, dient u de teksten uit te voeren op het object met ID <XWP_TRASHCAN>.

EMPTYTRASH=IMMEDIATE

Leegt de prullenbak. Er wordt geen bevestiging gevraagd.
Dit is niet echt een "instelling", maar meer een opdracht die uitgevoerd moet worden op de prullenbak. Wees er op bedacht dat de opdracht pas klaar is als de prullenbak geleegd is.

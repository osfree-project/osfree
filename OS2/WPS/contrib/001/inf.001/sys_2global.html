With the &xwp; Global Settings, you can make &xwp; behave differently. Here you will find settings which affect &xwp; in general, such as its language support, as well as settings which will affect your computer as a whole. This is why they're called "global settings", as opposed to individual folder settings, which are explained on a separate page.

The major features of &xwp; can be enabled or disabled in the new "&xwp; Setup" object. Depending on the settings on the "Features" page, certain parts of &xwp; will be completely disabled. This might be useful if you are only interested in certain features or if you have problems with others.
New "&xwp; Setup" object

The detailed settings for the various &xwp; features however are in the "Workplace Shell" object, which you will also find in your "System setup" folder.
New "Workplace Shell" object

Note that if you disable features in "&xwp; Setup", some settings pages in "Workplace Shell" will not be shown if they don't make sense without a feature being enabled.

The different settings pages are not explained in detail here. You will find extensive help by pressing F1 or the "Help" button, which exists on every page.

This chapter offers you various reference information on &xwp; and also some on the WPS which is otherwise hard to get:
  • WPS class hierarchy: a diagram of the most important WPS classes
  • Desktop object IDs: a collection of object IDs which should be valid on most systems
  • WPS setup strings: setup strings which can be used on Desktop objects
  • WPS menu ID's: a collection of menu ID's that can be used with the MENUITEMSELECTED setup string
  • &xwp; INI keys: a collection of OS2.INI keys that &xwp; uses

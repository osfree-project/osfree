The "Copy" menu item will make a copy the selected objects in a different target location.

A window will open that allows you to specify the folder where the objects should be copied to. The window has several tabs to specify the target folder in various ways.

Remarks:

  1. If the target folder is already open, you can copy the objects more quickly by simply dragging and dropping them onto the target folder. Hold down the Ctrl key while dragging to make sure the objects are not moved (instead of copied).

  2. Copying a shadow does not copy the original object, but the shadow only.

  3. If you are copying files between different drives, be aware that there might be file-system restrictions. See "Restrictions when copying and moving files" for details.

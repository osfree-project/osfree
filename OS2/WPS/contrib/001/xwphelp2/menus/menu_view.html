The "View" submenu contains several items that allow you to change frequently used folder view settings quickly without having to browse through the folder's properties notebook.

To disable these menu items, uncheck the "Layout items" setting on the "Menu" pages in the "Workplace Shell" object.

The following icons are available, depending on the folder's current view:

  • "Small icons" is available in Icon and Tree views and allows you to toggle whether the view should display regular sized or small icons. Depending on the current view, this corresponds to the one of the settings on the folder's "View" notebook pages. Note that the Details view will always use small icons.

  • "As placed", "Multiple columns" and "Single column" are available in Icon views only and allow you to switch between the various types of folder Icon views. This corresponds to the setting on the folder's "Icon view" notebook page.

  • On Warp 4, "Menu bar" allows you to quickly enable or disable the folder's menu bar. This will affect all folder views. This corresponds to the setting on the folder's second "Menu" notebook page.

  • You can also enable or disable the folder's status bar setting. This corresponds to the setting on the folder's "View" notebook page. Note that status bars are always disabled for the Desktop.

This setting determines whether &xwp; will collect all the icons when displaying folder content menus, similar to the way the &warpcenter; displays its submenus.

Note that enabling this setting will slow down the folder content functions, because collecting and displaying all the icons in a folder takes some additional time and memory.

This setting applies to all folder content menus on the system. It also affects the folder content menus that are added because you have declared a folder to be your "favorite" folder on the "View" page in its properties notebook. Besides, it will affect the menus for object button widgets in the &xcenter;.

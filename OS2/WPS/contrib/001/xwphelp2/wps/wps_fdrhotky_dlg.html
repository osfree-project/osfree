This dialog allows you to define a folder hotkey for a particular folder command.

Perform the following steps:

  1. Click into the "Hotkey" entry field. Press the key combination you would like to assign to the new folder action. Note that the entry field will only react if &xwp; considers the keys a valid hotkey combination. For example, neither A nor Shift-A would be valid, but Ctrl+A or Ctrl+Shift+A would be.

  2. From the "Parameter" drop down, select the folder action that should take place when your key combination is pressed in a folder.

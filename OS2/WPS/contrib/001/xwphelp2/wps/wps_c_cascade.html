This determines whether &xwp; will give its new submenus the "conditional cascade" style.
  • When this box is checked, &cfgfdr; submenus will look like this: Clicking on the submenu title will automatically select the default menu item in the submenu.
  • When this box is not checked, &cfgfdr; submenus will look like this: There will be no default menu items in the submenus.

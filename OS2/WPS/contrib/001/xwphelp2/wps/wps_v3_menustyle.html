In the "Menu style" group, you can set whether the system should use long or short object menus. If you select "Short menus", all functions that can also be accessed via mouse or keyboard will be removed. This applies to items like "Copy", "Move", "Create shadow" etc. (You can override this setting individually in each folder's properties notebook.)

Note that on the following notebook page, you can still select in detail which menu items should be visible for object menus.

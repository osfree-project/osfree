The "Paste" dialog allows you to create a new file in the current folder that receives the current contents of the &os2; clipboard.

In the dialog, the "Preview" field shows you the current contents of the clipboard. This is what the new file will contain.

In the "Object title" field, enter a name for the object that you want to create. The dialog remembers the last 10 object titles that you have entered in the list under the entry field.

From the list under the "Format" field, select the format that shall be used for copying the clipboard data to the new file. Depending on the application that produced the current clipboard contents, there may be several formats available.

From the list under the "Object class" field, select the Workplace Shell Class for the new object. The contents of the list depend on the format that is selected in the list above.

After you have made your selections, press "OK" to have the new object created.

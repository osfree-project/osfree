This page allows you to configure various startup settings. All these settings will not take effect until the next Desktop restart.

Note that the &xwp; Startup folder has its own notebook page to configure its behavior once it has been created (see below).

For a detailed explanation of each field, select from the list below:

  • Workplace Shell boot logo
  • Write WPSTART.LOG file
  • Show class initialization
  • Enable NumLock at startup
  • Create &xwp; Startup folder

"Add Snap to grid" will add a "Snap to grid" menu item to this folder's &popmenu; (for Icon views only, and only if "Always maintain sort order" is disabled for this folder).

Changing this setting overrides the default setting specified on the "Snap to grid" page in the "Workplace Shell" object.

This setting is disabled if "Snap to grid" has been disabled altogether in the "&xwp; Setup" object.

On this notebook page you can configure some view settings individually for every folder. This allows you to override some of the global settings specified in the "Workplace Shell" object. Initially, the settings will be the same as with the global settings; if you change anything here, this will be remembered for this folder only, and changes to the global settings will not affect this folder any more.

Changing the settings will take effect immediately.

  • If a setting is checked, the respective setting will always be enabled for this folder, no matter what you specify in the global settings.
  • Reversely, if a setting is unchecked, the respective setting will always be disabled for this folder.
Press "Undo" to restore the settings that were in use before you opened this notebook page.

Press "Default" to make this folder behave according to the global settings in the "Workplace Shell" object. This will remove the data which was remembered for this folder.

For a detailed explanation of each field, select from the list below:

  • Favorite folder
  • Quick open at Desktop startup
  • Show full path in folder title
  • Add Snap to grid
  • React to folder hotkeys
  • Display status bar
Additional information can be found on the corresponding help panels for the respective pages in the "Workplace Shell" object.

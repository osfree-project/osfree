This is xfix, the &xwp; File Handles Fixer.

What xfix can do for you:

  • Dump the contents of the WPS's file handle data blocks from the OS2SYS.INI files into a PM container. This is what you are seeing now.

  • Allow you to remove each handle individually.

  • Allow you to remove folder position entries which have been wrongly assigned.

  • Move lost abstract objects to the Desktop.
Compared to Henk Kelder's formidable CHECKINI program (from the WPTOOLS package), xfix gives you more control, but also requires more attention and knowledge.

Thanks go out to Henk Kelder again for finding out how the handles work in the first place. xfix is partly based on his work.

Warning: This program allows you to even remove file handles that the WPS relies on, such as the handles for the Desktop. xfix will allow you to make any changes to the file handles without preventing changes that should not be made. Only use this program if you know what you're doing. Read through all the following before using this.

In general, you are safe until you select "Write back to OS2SYS.INI" from the "xfix" menu. Only this will actually modify your system.

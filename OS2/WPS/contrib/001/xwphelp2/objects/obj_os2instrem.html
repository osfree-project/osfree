The "Install/Remove" folder contains objects that allow you install additional software on your computer. Initially, this contains objects to install additional &os2; components. You can also use these objects to remove &os2; components that you have previously installed.

You find this object in the &link_sysfdr; on your Desktop.

Double-click on the folder icon to display the objects in the folder.

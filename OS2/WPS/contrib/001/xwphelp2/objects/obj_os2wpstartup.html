The "Startup" folder is processed when the system is started. All objects contained in this folder are opened automatically at every system startup.

You find this object in the &link_sysfdr; on your Desktop.

By placing &link_progobjs; into this folder, you can automatically start programs that you frequently use.

The "Startup" folder is most frequently used with programs that are to be started during system initialization but might go away shortly after. For example, a batch file that starts a network server or requester is a typical use.

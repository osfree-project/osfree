The &xwp; trash can collects all the objects that you have recently deleted. This allows you to recover objects that you deleted accidentally. See "Deleting objects" for details.

If you open the trash can (it supports only the Icon and Details views), you can see all the objects that have been moved into it.

To restore a trash object, either open its &popmenu; and select "Restore object", which will move this object back to its original location, or drag it out of the trash can and drop it onto any folder, which will restore the object in that location instead.

To permanently delete a trash object, select "Destroy object" from its &popmenu;.

If the trash can is not empty, the trash can's &popmenu; offers you an additional menu item called "Empty trash can", which will permanently delete all the objects which have been moved into the trash can.

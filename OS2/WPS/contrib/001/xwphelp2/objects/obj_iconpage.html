The "Icon" page exists in the properties notebooks of almost every object and allows you to change an object's title, icon, and other basic object settings.

&xwp; replaces the WPS's original "Icon" pages with a new, enhanced version if you have enabled the respective setting on the "Features" page in the "&xwp; Setup" object. Disable that setting to get the original page back.

Press "Undo" to restore the settings that were in use before you opened this notebook page.

Press "Default" to restore the object's settings to default values, which are normally determined by the object's class.

For a detailed explanation of each field, select from the list below. Note that depending on the object's class, not all of these fields may be available on the page:

  • Title
  • Template
  • Lock in place
  • Object hotkey
  • Current icon
  • Details

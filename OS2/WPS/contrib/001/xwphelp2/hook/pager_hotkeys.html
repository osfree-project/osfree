If "Arrow hotkeys switch desktops" is checked, you can switch virtual desktops using the arrow keys on your keyboard. You can specify the modifier keys which must be held down in addition to the arrow keys. &xwp; does not allow you to disable all the modifiers because this would prevent you from using the arrow keys in the normal way. The default setting is to have Ctrl and Alt as the modifier keys.

Note that the arrow hotkeys work even if you have disabled global hotkeys in the "&xwp; Setup" object.

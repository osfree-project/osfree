In the Workplace Shell, every object has associated with it a &popmenu; that displays the options that are currently available for an object.

To open the &popmenu; of any object, move the mouse pointer over it and press mouse button 2. A menu will open with several options in it.

Exacly what options are displayed will vary depending on the object. The &popmenu; only displays options that can presently be used with the object. For example, objects that cannot be deleted will not have the "Delete" menu item. As another example, &link_folders; usually have a lot more menu items than other objects.

The &popmenu; can also be used to open an object's properties notebook by selecting the "Properties" menu item:

In addition, you can select to open a specific view from the object's "Open" submenu. This is helpful for folders, for example, to have a view opened that is different from the default view that gets opened when you double-click on the folder.

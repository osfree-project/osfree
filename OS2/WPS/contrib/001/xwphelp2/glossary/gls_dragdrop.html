Dragging and dropping allows you to perform operations on an object when more than one folder is involved. This is a quick way of copying or moving objects or creating shadows.

Within a single folder view, you can drag and drop objects to arrange and reposition them.

Perform the following:

  1. Place the mouse pointer over an object.

  2. Press mouse button 2 and keep holding it down.

  3. Move the mouse to the target of the operation while holding down mouse button 2 all the time. The object moves around the screen with the mouse pointer:

    This is called dragging an object.

  4. When the mouse pointer is over the target, release mouse button 2. This is called dropping an object. The desired operation will be initiated.
Depending on the folders that are involved, if you do not hold down any keys while performing the drag and drop, a default operation will be performed. To enforce copy, move, or create shadow, hold down either the Shift or Ctrl key or both. See "Copy, move, and shadows overview" for details.

If several objects are selected and you begin dragging one of them, the operation will be performed on all of them together.

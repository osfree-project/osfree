Substitution variables are specified as %0 through %9. When these variables are encountered in a batch or command file, the command processor replaces them, in sequence, with the arguments that were given to the file.

&os2; reads the percent sign (%) as a command rather than as a regular input character. If you intend to use the percent sign as an input character, you must specify it twice (e.g. as in ABC%%.IN).

If you need more than nine variables, you can use the SHIFT command.

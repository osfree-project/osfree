• To select an object in a folder, click on the object with mouse button 1.

  • To open the default view of an object, double-click on it.

  • To open the &popmenu; of a contained object, click on the object with mouse button 2. To open the &popmenu; for the folder itself, click on empty space in the folder container.

  • To copy or move an object, or to create a shadow for it, drag and drop it to a new folder with mouse button 2.
Note that if several objects are selected, all selected objects will be affected by the mouse operation.

The default sort criterion is painted with a check mark in the "Sort" menu and will be used by default if you have "Always maintain sort order" enabled.

You can specify the default criterion

  • globally, on the "Sort" page in the &link_wpsobj;;

  • individually for each folder on the "Sort" page in its properties notebook;

  • individually for each folder by holding down the "Shift" key while clicking on one of the sort criteria in the "Sort" menu.
See "Sort Criteria" for a description of the available sort criteria.

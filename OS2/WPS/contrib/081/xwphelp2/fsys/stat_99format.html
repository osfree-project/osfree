Several status bar codes which display byte counts require an extra format specifier character after the first two code characters. For example, with the $sX code (for total size of selected objects), X is the format specifier, and must be one of:
  • b: display the value in bytes.
  • k: display the value in kBytes, i.e. bytes / 1000
  • K: display the value in KBytes, i.e. bytes / 1024
  • m: display the value in mBytes, i.e. bytes / 1,000,000
  • M: display the value in MBytes, i.e. bytes / 1,048,576
  • a: display the value in bytes/kBytes/mBytes, depending on the size of the value
  • A: display the value in bytes/KBytes/MBytes, depending on the size of the value

Compared to other objects, the &xwp; trash can has a special "Icon" page. Note that you cannot use this page to change the icons which are used by the trash can. As opposed to other folders, the trash can uses two different icons depending on whether objects are currently in the trash can.

If you have icon replacements enabled in the "&xwp; Setup" object, you can change the trash can icons as described in the &xwp; User Guide.

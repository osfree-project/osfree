The &xcenter; "Window list" widget shows you the contents of the PM window list as buttons.

If a window is currently active, its button is displayed as if it were pressed down.

You can also click on the window buttons. Depending on whether the window is currently active and/or minimized, this will activate or minimize the application which the button represents. Minimizing works only for frame windows which have a minimize button though.

Note: If the window button represents a full-screen session, clicking on the button will switch to that session, away from the &os2; Desktop.

Note that you can drag and drop fonts and colors onto the window list.

You can also configure the window list widget to suppress the display of certain windows ("filtering"). Right-click inside the widget and select "Properties". A dialog will pop up which allows you to filter out any windows that you don't want to see.

The &xwp; Hook must be enabled for this widget to work (on the "Features" page of the "&xwp; Setup" object).

This dialog is used for configuring HPFS.IFS.

For introductory information about file systems, select from the following list:

  • File systems
  • File-system caches
Press "Default" to have the &os2; default settings displayed on the notebook page.

By pressing "OK", the selections you made will be translated into the "Parameters" field on the "Drivers" page in the "&os2; Kernel" settings notebook. This does not change your &cfgsys; yet. To do that, press "Apply" on the "Drivers" page after pressing "OK" in this dialog.

For a detailed explanation of each field, select from the list below:

  • Cache size
  • Autocheck

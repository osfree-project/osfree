With the text fields in the "Content of status bars" group, you can determine what will be displayed in the status bars.

&xwp;'s status bars operate in three different modes, depending on whether no object, just one object or several objects are currently selected in the folder. For each of these three modes, you may enter any text you like into the respective text field.

  • The keys in the field "No object selected" determine what information will be displayed if no object at all is currently selected in a folder.

  • In "one-object mode", &xwp; supports specific status bar information depending on the class of the currently selected object. For example, if you select a program object, the information will be different compared to a selected data file object or folder.

    Press the "Select class..." button to select the WPS class for which you wish to set status bar information. The codes you then enter into the entry field affect objects of this class only.

    If the "Dereference Shadows" setting in this group is enabled, &xwp; will, if a single shadow object is currently selected, not display the information for the shadow object (which is pretty useless), but for the object which the shadow points to instead.

  • The codes in the field "Several objects selected" determine the information to be displayed when more than one object is currently selected.

    If the "Dereference Shadows" setting in this group is enabled and multiple objects are selected, among which are shadows, &xwp; will use the information of the object which the shadows point to for computing the "selected objects" sizes. Of course, this may give you a size for the selected objects which can be larger than the size of the objects in the folder, since the shadowed objects don't really belong to the folder.

Note that all changes take effect immediately so that you can view the results in an open folder while you experiment with the different status bar codes.

&xwp; uses special codes to display variable Desktop object information. All codes start with the "$" character. If &xwp; does not recognize a certain text you enter here as a status bar code, it will simply be copied into the status bar display, so you can also add informative text here.

The codes that can be used in the status bar content descriptions depend on the mode that the status bar currently operates in. For detailed descriptions of the various codes, select from the following list:

  • Common codes for all three modes
  • Common codes for one-object mode only
  • Codes for WPFileSystem objects in one-object mode
  • Codes for WPDisk objects in one-object mode
  • Codes for WPProgram objects in one-object mode
  • Codes for WPUrl objects in one-object mode

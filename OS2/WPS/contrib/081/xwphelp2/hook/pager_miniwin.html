If "Mini-windows" is checked, the &pgr; window will contain a shrinked copy of all windows which are currently visible on any of the virtual desktops. If "Show window titles" is also enabled, those mini windows will contain the respective window titles also.

If the "Mini-window mouse actions" setting is also enabled, you can click into the &pgr; window to have certain actions performed on any window on any virtual desktop. See "&pgr; mouse actions" for details.

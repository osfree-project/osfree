If "Flash to top" is enabled, the &pgr; window will temporarily move itself on top of all other windows when desktops are switched. This happens either if you move to another desktop using the desktop hotkeys or if a window gets activated which is on another desktop.

This setting allows you to preserve space on your screen because the &pgr; window will only show up when it's needed.

With the "Delay" setting you may specify (in seconds) how long the &pgr; window will show before it hides again.

Note: Since the &pgr; window will keep disappearing if this setting is enabled, you can still see the &pgr; window when the mouse is moved to a screen corner or border via a "special function" in the &xwp; "screen borders" feature.

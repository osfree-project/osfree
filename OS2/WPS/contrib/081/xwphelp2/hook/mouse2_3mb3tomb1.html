The "MB3 click action" drow-down list allows to assign a specific action to mouse-button-3 (MB3) clicks:
  • If "AutoScroll" is selected, then all MB3 clicks start the autoscroll mode if the mouse pointer is over a scrollable area. While in auto-scroll mode, moving the pointer scrolls the area. The farthest the pointer is from its initial position, the fastest the area scrolls. Clicking any mouse button exits the autoscroll mode.

    Note: This does not interfere with the "MB3 drag scrolling" settings below that determine what happens when you drag the mouse while mouse button 3 is down. You can have AutoScroll enabled and drag-scrolling disabled, and vice versa.

  • If "Double click" is selected, then all MB3 clicks will be converted to mouse-button-1 double clicks. This allows you, for example, to single-click on any Desktop object with button 3 to have the object opened.

  • If "No conversion" is selected, then MB3 click actions are disabled. No translation occurs for MB3 clicks, so applications that define a specific action for MB3 clicks will recognize them.

  • If "Push to bottom" is selected, then all MB3 clicks over a window will push that window to the bottom of the Z-order stack. This can be handy if your Desktop is crowded.
Note that MB3 clicks are only converted if the mouse is not moved while mouse button 3 is held down.

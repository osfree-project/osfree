&xwp; extends the ways in which you can create new objects. Among others, the following options exist: &xwp; allows the following ways of creating new objects:
  • Open the &link_tempsfdr; and drag a new object off a template.

  • To create a new object of the same type as an existing object, open the &popmenu; of that object and select a choice from the "Create another" menu.

  • If you find yourself trying to create the same type of object frequently, you might find the "Create new" menu more convenient. This menu always contains the same list of templates, which is taken from the &xwp; &cfgfdr;. By default, frequently used object types such as folders are in that menu, but you can change that list yourself.

    See "Creating objects with Create New" for more.

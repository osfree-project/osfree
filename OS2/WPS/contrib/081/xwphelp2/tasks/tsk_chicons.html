With &os2;, every object has an icon, a picture that represents the object and makes it easy to recognize what the object can do. Most importantly, icons are displayed in folder views.

The initial icon of an object is either determined automatically from the object's class or, in the case of programs, from the program file itself.

You can change almost every object's icon manually. Perform the following steps:

  1. Open the properties notebook of the object.

  2. Select the "Icon" tab from the list of pages displayed at the top of the notebook.

  3. Drop any icon onto the "Icon" field, or select one of the buttons on the page. Press the "Help" button on the page for details.
Note that it is possible to create objects with a non-default icon from the beginning. Many installation programs will also create objects with non-default icons. See "Setup strings" for details.

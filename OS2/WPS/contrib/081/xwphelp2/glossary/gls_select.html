In order to work with an object, you must select it first.

To select an object, do the following:

  1. Move the mouse pointer over the object.

  2. Press mouse button 1 once. This is called clicking.
The object will change its color to show that it is now selected.

Every time you click on an object, a previously selected object is deselected by default. In order to select several objects, you can hold down the Ctrl key while clicking on objects. In that case, previous selections are not removed.

Alternatively, hold down mouse button 1 and move the mouse pointer. This will select all objects under the mouse and is called swipe select.

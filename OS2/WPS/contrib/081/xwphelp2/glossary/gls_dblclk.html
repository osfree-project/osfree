"Double-clicking" on an item means clicking twice with mouse button 1 while the mouse is over that item.

Double-click on an object to have it opened in its default view. If you double-click on a folder, a folder view opens that displays the objects in it.

Note that you can set the double-click speed (that is, the amount of time that may pass between two clicks before &os2; no longer considers two clicks as a double click) in the "Mouse" object in the &link_syssetupfdr;.

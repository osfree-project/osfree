"Enhanced Editor" is a powerful editor program that allows you to work on multiple files at the same time. While the &os2; &eexe; is mainly used for quickly viewing or editing small files, the enhanced editor can manage more complex projects. It can do syntax highlighting and many other advanced features, so many programmers use it as their favorite programming editor.

For detailed help, start the enhanced editor and select the "Quick Reference" item located under the "Help" menu bar choice.

You can also start the Enhanced Editor by typing EPM at a command prompt and pressing Enter.

The second "Icon" page exists only in properties notebooks of folder objects and allows you to change a folder's animation icon.

&xwp; replaces the WPS's original "Icon" pages with a new, enhanced version if you have enabled the respective setting on the "Features" page in the "&xwp; Setup" object. Disable that setting to get the original page back.

The animation icon is displayed for a folder object if the folder currently has an open view. By default, folders have an "open folder" animation which makes it more obvious that the folder is currently open, but you can change this animation icon to anything else.

Press "Undo" to restore the settings that were in use before you opened this notebook page.

Press "Default" to restore the folder's animation icon to default values, which are normally determined by the object's class.

The "Current icon" field works exactly as with the respective field on the first "Icon" page, except that it only influences the animation icon of the folder. See Current icon for more instructions.

The "Registry Editor" allows you to look at the system registry.

This object resides in the &link_syssetupfdr;. Open the Desktop's &popmenu; and select "&syssetup;" to have that folder opened quickly.

The system registry is a central database that &os2; provides for applications to conveniently store persistant data. &os2; also uses this database itself to store system configuration and the state of the Desktop between reboots.

The system registry is composed of the OS2.INI, OS2SYS.INI, USER.DAT, and SYSTEM.DAT files.

You can also use the Registry Editor to add, change, or delete key and values in the registry. However, you should take great care when doing so because much of the registry is critical to the system's operation, and making invalid changes can render your entire system unusable.

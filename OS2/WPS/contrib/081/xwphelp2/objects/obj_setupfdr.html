The "&syssetup;" folder is a special folder created by &os2; to contain objects for system configuration of all sorts.

In this folder, you will find objects for setting system colors and fonts, mouse and keyboard settings, locale support, the system time, and many more.

Double-click on the "&syssetup;" folder icon to display those objects. If you are unsure what an object does, select it and press the F1 key to open a window with help about it.

You find this folder in the &link_sysfdr; on your Desktop. Note that the Desktop's &popmenu; has a shortcut "&syssetup;" menu item that will open this folder as well.

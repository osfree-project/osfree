&xwp; is an open source project. As opposed to previous XFolder versions, the source code now entirely resides on the OS/2 Netlabs server.

We use the Concurrent Versioning System (CVS) to cooperatively work on the source code. Basically this means that there is a common source tree on an Internet server (in this case, the one of Netlabs) which allows multiple people to work on the sources at the same time. CVS is then responsible for keeping all the changes in sync. At http://www.xworkplace.org, you will find instructions where to find the sources, including detailed instructions about how to use CVS.
